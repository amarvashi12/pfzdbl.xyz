WoWonder Desktop - A Windows Messenger For WoWonder


WoWonder Desktop is a windows chat application for WoWonder social network, with WoWonder Desktop users can chat together on their PCs using our new application, now chat is faster, easier, and more fun ! WoWonder Desktop is easy, secured, and it will be regularly updated.

Installation

The installation is pretty easy, please follow the steps below:

  1. WoWonder PHP Social Network Script, you can get it from Here.
  2. Download Visual Studio Professional 2015
    • Download Link Here
    • Video Tutorial
  3. Download Microsoft Visual Studio 2015 Installer Projects Here.
  4. Unzip the WoWonderDesktop archive, extract it to new folder, and then open the folder.
  5. In the main folder you will find the solution (Name:WowonderLight Type: Microsoft Visual Studio Solution) double click on it and wait till everything is loaded.
  6. In the Visual Studio menu bar go to Build > Clean Solution, click it and wait till it finish.
  7. In the Visual Studio menu bar go to Build > Build Solution, click it and wait till it finish building your project.

How can i modify the style?

  • You can change it via your WoWonder web script admin panel.
